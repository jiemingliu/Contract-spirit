jQuery内参

这个jQuery的意思是js+Query，也就是js的查询的意思，结合英文意思，很好理解的是jQuery的宗旨就是帮助你更快的查询元素。

  • 1、jQuery实际上还是js代码，这个可以去打开源码查看就明了了，jQuery只是对原生js进行了一层封装而已。比如说DOM中的查询ID元素，getElementById()方法，在jQuery中用$("#id")表示，但是实际上jQuery的源码中还是用的getElementById()，所以也就是说除了极个别特殊的功能，大部分所谓的jQuery方法，都可以用原生的js代码实现.所以在学习jQuery的过程中结合之前学的js基础，应该是很容易理解的，概念原理之类的都是一样的，只是换了一种写法而已，甚至你完全可以不学jQuery，代码都用js实现也是可以的，但是，这里，我们还是学一下，毕竟，他对原生js封装一下之后，写起来要快一些。
  • 2、jQuery对象和DOM对象虽然不是同一种对象，但是，内涵是一样的
  • 3、关于子选择器，后代选择器，兄弟选择器，父选择器等等这些层级选择器，可以把一个文档想像成一个家族去理解，根节点，也就是document是祖先，下面的都是繁衍出来的后代子子孙孙，这些后代的关系错综复杂，跟贾府一样复杂，但是再怎么复杂，都是从根节点上繁衍出去的，都可以找到对应的亲戚关系，也就可以利用这个亲戚关系去进行查找。
  • 4、包括上面的层级选择器，还有很多选择器譬如基本选择器，内容选择器，可见性选择器，属性选择器(这个可以结合正则表达式理解，正则表达式实际上就是一个用来过滤字符串的规则而已，不要想复杂了)等等选择器的学习方法，这些选择器的用法在学的时候不需要强行记住，重在理解，要知道的是在之后的工作过程中，比如说，当你要查找一个元素的第三个子元素的时候，你知道要使用层选择器去选择就行，不知道怎么写，就回过头来翻资料了。甚至关于这些选择器的部分，你可以直接快速浏览一遍过去，因为大部分选择器根本用不到，常用的永远只是小部分，等到常用的实在不够解决问题的时候，再去看一下生僻的选择器有没有能解决问题的
  • 5、有关DOM操作，各种插入节点，删除节点，修改节点等等方法，记住四字宝典，增删改差，你就知道他们所有的那么多方法都是完成这四个字的功能
  • 6、有些小节的代码在教程里面验证是没有效果的，第一反应是copy到本地验证效果。jQuery基础(四)—动画篇，都需要在本地验证。
  • 7、关于jQuery插件的讲解，不需要完全弄懂 Validation Plugin 的用法，只要搞清楚插件是什么，为什么有插件的存在就可以了。这主要是为什么解决问题提供了一种思路，当我们遇到一些特殊的需求的时候，我们用常规的方法不能解决问题，或者说用常规方法不方便的时候，我们可以寻求其他的解决办法，也许办法就藏在某种插件里面，异常的方便快捷
  • 8、关于jQuery的那么多用法，知道有这些用法就行，完全不用在意细节，也就是说不需要把怎么使用搞的一清二楚，知道有这些方法就行
  • 9、jQuery的动画，第1234章都是为了引出第五章的 animate 的用法，基本上会animate动画就可以了，animate这个jQuery函数有两个用法
    第一种, .animate( properties ,[ duration ], [ easing ], [ complete ] ),当一个函数的参数列表有用中括号表示的时候，表示这个参数是可选的，意思是调用这个函数的时候，可以传这个参数，也可以不传。
    例如这个3s透明度变成0，
    				$(element).animate({   
    				opacity:0
				},3000)
    例如这个3s透明度变成0.5，宽度变成300px，动画结束后弹出完成的窗口
    				$(element).animate({
					opacity:0.5,
					width:300px
				},3000,function(){alert("动画完成")});
    第二种，.animate( properties, options ) ，第一个参数properties同上，第二个参数 options 可以设置参数，或者获取动画执行中的一些状态值,至于有哪些状态值可以参考教程
    例如这个
    				$aaron.animate({
                	height: '50'
            		}, {
                	duration :2000,
                	progress: function(now, fx) {
                    	$aaron.text('进度:'+arguments[1]);
                }
            })
